# Webseite des Code Club Waghäusel

**Hinweis: Der Code Club Waghäusel befindet sich im Aufbau und startet im Herbst 2026**

Der Code Club Waghäusel ist ein **kostenloses** Programmier-Angebot für Kinder und Jugendliche im Schulalter. Die Code Clubs sind Teil der Raspberry Pi Foundation, einer gemeinnützigen Organisation mit der weltweiten Mission, jungen Menschen durch Informatik und digitale Technologien zu helfen, ihr volles Potenzial zu entfalten.

Das Motto lautet: Lernen durch Machen. Mit kostenlosen Materialien und der Unterstützung von Mentoren erhalten junge Menschen die Möglichkeit, kreativ zu sein, neue Fähigkeiten zu entwickeln und sich mit Gleichgesinnten zu vernetzen.

Weltweit sind bereits über 2 Millionen Kinder und Jugendliche in mehr als 100 Ländern Teil der Code-Club-Gemeinschaft.
